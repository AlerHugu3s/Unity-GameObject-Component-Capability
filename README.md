# UnityGCC

这是一个Unity项目，包含基本的游戏开发功能。

## 项目结构

- `Assets/` - Unity资源文件夹
  - `Scenes/` - 场景文件
  - `Scripts/` - C#脚本文件
    - `Capabilities/` - 功能模块
    - `Components/` - 组件脚本
- `ProjectSettings/` - Unity项目设置
- `Packages/` - Unity包管理

## 开发环境

- Unity 2022.3 LTS 或更高版本
- Visual Studio 2022 或 Visual Studio Code

## 开始使用

1. 克隆此仓库
2. 在Unity Hub中添加项目
3. 打开项目并开始开发

## 注意事项

- 请确保在提交代码前测试所有功能
- 遵循Unity的最佳实践
- 保持代码整洁和可读性
